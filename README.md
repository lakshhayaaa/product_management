# Product Management System

A full-stack **Product Management System** built using **FastAPI** for the backend and **React (Create React App)** for the frontend.  
This project demonstrates **basic CRUD (Create, Read, Update, Delete)** operations using **SQLAlchemy ORM** and **PostgreSQL** as the database.
